# POE_PART_2 
Cybersecurity Awareness Chatbot
Overview
The Cybersecurity Awareness Chatbot is a C# WPF application developed to educate users about common cybersecurity threats and safe online practices. The chatbot provides information on topics such as phishing, malware, password security, and safe browsing while maintaining interactive conversations with users. 

 Features:
User-friendly graphical user interface (GUI)
Username validation
Personalized user greetings
Voice greeting functionality
Cybersecurity topic recognition
Randomized chatbot responses
Sentiment detection and supportive responses
Memory feature for storing user interests
Follow-up conversation handling
File-based data storage
Error handling for invalid input

Cybersecurity Topics Covered:
Phishing
Malware
Password Security
Safe Browsing
Online Safety

Technologies Used:
C#
WPF (Windows Presentation Foundation)
.NET Framework
File Handling
Regular Expressions (Regex)
Dictionaries and Collections
Object-Oriented Programming Principles

Project Structure:
MainWindow.xaml – User interface design
MainWindow.xaml.cs – Application logic
Voice_Greeting.cs – Voice greeting functionality
UserData Files – Stores user information and preferences

How to Run:
Open the solution in Visual Studio.
Build the project.
Ensure the voice.wav file is included in the output directory.
Run the application.
Enter a valid username and begin chatting with the cybersecurity chatbot.
