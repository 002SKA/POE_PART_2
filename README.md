# POE_PART_2 
Cybersecurity Awareness Chatbot
Overview
The Cybersecurity Awareness Chatbot is a C# WPF application developed to educate users about common cybersecurity threats and safe online practices. The chatbot provides information on topics such as phishing, malware, password security, and safe browsing while maintaining interactive conversations with users. 

 Features:
User-friendly graphical user interface (GUI)
Username validation
Personalized user greetings
Voice greeting functionality
Cybersecurity topic recognition
Randomized chatbot responses
Sentiment detection and supportive responses
Memory feature for storing user interests
Follow-up conversation handling
File-based data storage
Error handling for invalid input
